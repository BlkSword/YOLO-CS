然后是对于YOLO相关训练的建议：

1、首先要的是足够的负样本。YOLO官方提供的模型已经是有一定识别基础的，但是很显然。cs中的光影、影子等会引起模型的误判（甚至地图），所有说训练时要给足够多的负样本让模型记住

2、其实是小目标的识别。在尝试过程中，小目标的识别一直是模型不敏感的地方，需要更多的训练样本进行训练，以强化这一能力

3、YOLO的设计太巧妙了，以至于在低端算力下依旧可以快速正常的识别。对于模型的选择，选择检测和姿态估计两个模型，二选一。个人认为检测模型会更轻松一些，因为标注只需要框+标注。而姿态估计的样本则更复杂，要标注。。17个点。

4、关于识别+自瞄。使用检测模型，然后自瞄的话可以采用中线原则，识别框的中线的上中下三个区域的锁定。通过特定区域的识别（更快），通过虚拟鼠标之类的来控制鼠标，并进行自瞄。
